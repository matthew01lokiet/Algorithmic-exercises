<h1 align="center">To Lower Case</h1>

## Description:
Implement function ToLowerCase() that has a string parameter str, and returns the same string in lowercase.

## Examples:

Example 1:

```
	Input: "Hello"
	Output: "hello"
```

Example 2:

```
	Input: "here"
	Output: "here"
```

Example 3:

```
	Input: "LOVELY"
	Output: "lovely"
```
