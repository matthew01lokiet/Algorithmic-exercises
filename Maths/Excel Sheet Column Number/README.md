<h1 align="center">Excel Sheet Column Number</h1>

## Description:
Given a column title as appear in an Excel sheet, return its corresponding column number.

## Examples:

Example 1:

```
Input: "A"
Output: 1
```

Example 2:

```
Input: "AB"
Output: 28
```

Example 3:

```
Input: "ZY"
Output: 701
```
