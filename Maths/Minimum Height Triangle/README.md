<h1 align="center">Minimum Height Triangle</h1>

## Description:
Given integers b and a, find the smallest integer h, such that there exists a triangle of height h, base b, having an area of at least a.

## Examples:

Sample Input 0

```
2 2
```

Sample Output 0

```
2
```
