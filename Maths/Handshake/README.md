<h1 align="center">Handshake</h1>

## Description:
At the annual meeting of Board of Directors of Acme Inc. If everyone attending shakes hands exactly one time with every other attendee, how many handshakes are there?

## Examples:

Sample Input

```
2
1
2
```

Sample Output

```
0
1
```
